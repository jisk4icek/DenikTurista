## 📦 Rychlý přehled

```
PaperMC 1.20+  |  Java 17+  |  Maven  |  SQLite  |  Zero Dependencies
```

Plugin poskytuje kompletní systém pro letní turistický event na Minecraft serveru:

- ✅ **Turistický deník** – interaktivní GUI pro každého hráče
- ✅ **Časové zámky** – známky dostupné až od určitého data/času
- ✅ **Milníky** – automatické odměny za sbírání
- ✅ **Airdropy** – limitované edice padající z nebe
- ✅ **Hologramy** – živý žebříček pomocí TextDisplay entit
- ✅ **Server-First** – odměny pro první hráče s dokončeným deníkem
